# Flutter 설치 및 실행 가이드 (Windows 11)

## 1. Flutter 다운로드

1.  아래 공식 사이트 접속\
    https://docs.flutter.dev/get-started/install/windows

2.  Windows용 Flutter SDK 다운로드 (zip 파일)

3.  원하는 위치에 압축 해제\
    예: `C:\src\flutter`

------------------------------------------------------------------------

## 2. 환경 변수(PATH) 설정

1.  시작 메뉴 → "환경 변수" 검색 → **시스템 환경 변수 편집**
2.  환경 변수 클릭
3.  시스템 변수 → Path → 편집
4.  새로 만들기 → 아래 경로 추가

```{=html}
<!-- -->
```
    C:\src\flutter\bin

5.  확인 후 PowerShell 재실행

------------------------------------------------------------------------

## 3. Flutter 설치 확인

PowerShell에서:

``` powershell
flutter --version
```

정상적으로 버전이 나오면 설치 완료.

------------------------------------------------------------------------

## 4. 필수 구성요소 확인

``` powershell
flutter doctor
```

문제 있는 항목이 표시되면 안내에 따라 설치 진행.

------------------------------------------------------------------------

## 5. 프로젝트 실행 방법

### 1) 프로젝트 폴더 이동

``` powershell
cd D:\coding\lab_note\frontend
```

### 2) 패키지 설치

``` powershell
flutter pub get
```

### 3) 웹으로 실행 (가장 쉬운 방법)

``` powershell
flutter run -d chrome
```

------------------------------------------------------------------------

## 6. 실행 가능한 기기 확인

``` powershell
flutter devices
```

------------------------------------------------------------------------

## 7. 자주 발생하는 문제

### flutter 명령어가 인식되지 않을 때

-   PATH 설정 확인
-   PowerShell 재시작

### No devices found

-   Chrome 설치 확인
-   Android Emulator 실행 확인

------------------------------------------------------------------------

## 완료 🎉

이제 Flutter 프로젝트를 Windows 11에서 실행할 수 있습니다.
